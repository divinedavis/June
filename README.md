# June

All your favorite places from around the world.

iOS app whose home screen replicates the Apple Maps look-and-feel as a starting point for a personal places companion. Built with SwiftUI + MapKit + CloudKit + WeatherKit, signed in with Apple.

## Stack

- iOS 17+, SwiftUI, Swift 5.9
- MapKit (dark-mode standard map, transit overlay, Look Around)
- CloudKit (private database) for favorites, recents, and the user profile picture
- WeatherKit for the weather + AQI pill
- Sign in with Apple

## Develop

```bash
brew install xcodegen          # one-time
xcodegen generate
open June.xcodeproj
```

Build from CLI:

```bash
xcodebuild -project June.xcodeproj -scheme June \
  -sdk iphonesimulator \
  -destination 'generic/platform=iOS Simulator' \
  build CODE_SIGNING_ALLOWED=NO
```

`June.xcodeproj` is generated by xcodegen and is gitignored — regenerate after editing `project.yml`.

## Ship to TestFlight

One-time setup (App Store Connect doesn't allow creating apps via API):

1. Create the app at https://appstoreconnect.apple.com/apps with Name **June**, bundle id **com.divinedavis.june**, SKU **june**, primary language **English (U.S.)**.
2. `cp scripts/asc-config.env.example scripts/asc-config.env` and fill in your key id / issuer / team / key path.
3. `python3 scripts/bootstrap-asc-app.py` — looks up the new ASC app id and writes it to `scripts/asc-config.env`.

Then, after every change:

```bash
scripts/ship-to-testflight.sh --auto-notes
```

`--auto-notes` formats release notes from `git log` since the last shipped commit. Each ship bumps `CURRENT_PROJECT_VERSION` automatically.
